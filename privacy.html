Privacy Policy

Last updated: March 2026

This application ("Analisi Spese e Patrimonio") is a personal, non-commercial web application that reads a CSV file from the user's Google Drive to display personal finance data in the browser.

Data Access

This application requests read-only access to Google Drive (drive.readonly) solely to locate and read the file main.csv inside the folder Cashew/main/ on the user's personal Google Drive account.

Data Collection

This application does not collect, store, transmit, or share any personal data. All data processing happens entirely within the user's browser. No data is ever sent to any external server.

Google OAuth Token

A temporary OAuth 2.0 access token is requested from Google to authenticate the user and access Drive. This token is used only during the active browser session, is never persisted to any storage, and is never shared with any third party.

Third-Party Services

The only third-party service used is the Google Drive API, accessed solely to read the user's CSV file. No analytics, advertising, or tracking services of any kind are used.

Data Retention

No user data is retained by this application. Closing or refreshing the browser tab clears all loaded data.
